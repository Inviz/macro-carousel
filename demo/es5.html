Basic usage

In order to make this demo work with older browsers, this demo is slightly different from the rest of the demos: (have a look at this page's source code)

  • the transpiled version of <macro-carousel> is used instead (see demo.js)
  • the set of babel-polyfills is added before importing anything else
  • the custom-elements-es5-adapter is run for browsers supporting CustomElements v1
  • webcomponents-lite.js is used instead of webcomponents-loader.js
